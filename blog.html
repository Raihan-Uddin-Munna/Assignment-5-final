1.What is the Document Object Model (DOM)?

=> The DOM is a programming interface for web documents that represents the structure of an HTML or XML document as a tree of objects, allowing scripts to interact with, modify, and update its content, structure, and style dynamically.

2.How do you select an element from the DOM?

=> We are select an element from the DOM using many method's

  • getElementById()
  • getElementsByClassName()
  • getElementBysTagName()
  • querySelector()
  • querySelectorAll()
    • 3.What is event delegation in the context of the DOM, and why is it useful?

    => Event delegation is a technique where a single event listener is added to a parent element to handle events for its current or future child elements. It works because events bubble up from the target element to its ancestors. It is useful as it reduces memory usage, improves performance, and simplifies managing dynamically added elements.

    4.How do you manipulate an element's attributes and styles using the DOM?

    => Here the attributes and style:

    Attributes

    • To get an attribute: `element.getAttribute('attributeName')`
    • To set an attribute: `element.setAttribute('attributeName', 'newValue')`
    • To remove an attribute: `element.removeAttribute('attributeName')`

      Style

    • To get a style: `element.style.propertyName` or `window.getComputedStyle(element).propertyName`
    • To set a style: `element.style.propertyName = 'newValue'`
    • To remove a style: `element.style.propertyName = ''`
